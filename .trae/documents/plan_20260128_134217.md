## 前端代码修复计划

### 1. 修复 axios 配置
- 移除 baseURL 中的 `/api`
- Vite 代理已经配置了 `/api` → `http://localhost:8080`

### 2. 修复响应拦截器
- 登录接口返回 `ResultMapStringString`，Token 在 `response.data.data.token`
- 其他接口数据在 `response.data.data`

### 3. 修复 user store
- 登录后正确获取 Token：`response.data.data.token`
- 获取用户信息：`response.data.data`

### 4. 需要调整的接口
| 接口 | 修复内容 |
|------|----------|
| 登录 | `response.data.data.token` |
| 注册 | `response.data.data` (userId) |
| 获取用户信息 | `response.data.data` |
| 获取偏好列表 | `response.data.data` |
| 获取库存列表 | `response.data.data` |
| 获取菜谱列表 | `response.data.data` |
| 获取菜谱详情 | `response.data.data` |

### 5. 修复后的响应处理

// 在拦截器中处理统一响应
